# Product Specification

## Overview

Ajaia Docs is a lightweight collaborative document editor built for teams who need simple, fast document workflows.

## Core Features

### Document Editing
- Rich text formatting with bold, italic, underline
- Headings (H1, H2, H3)
- Bullet and numbered lists

### File Import
- Upload `.txt` files as editable documents
- Upload `.md` files with heading and list parsing

### Sharing
- Share documents by email
- View or Edit permission levels

## Technical Stack

1. Next.js 14 (App Router)
2. TipTap rich text editor
3. Supabase (PostgreSQL + Auth)
4. Tailwind CSS

## Status

The project is currently in active development.
